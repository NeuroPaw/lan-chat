# 💬 局域网聊天室

一个为同一 WiFi/局域网 内设备打造的实时聊天应用。

## ✨ 功能特点

- 💬 **实时消息** - 使用 WebSocket 即时传递消息
- 👥 **在线用户** - 实时显示谁在线
- 🖼️ **文件分享** - 支持发送图片和文件
- 📱 **移动端适配** - 响应式设计，支持各种设备
- 💾 **消息历史** - 保留最近的消息记录
- 🔒 **纯本地** - 无需联网，离线也能用

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 或者直接运行
node server.js
```

## 📱 使用方法

1. 在你的电脑上启动服务器
2. 浏览器打开 `http://localhost:3000`
3. 同一 WiFi 下的其他设备用你的本地 IP 访问：`http://你的IP:3000`
4. 输入昵称开始聊天！

## 🔧 系统要求

- Node.js 14 或更高版本
- 所有设备必须在同一 WiFi/局域网内

## 📁 项目结构

```
lan-chat/
├── server.js          # Express + Socket.IO 服务器
├── public/
│   └── index.html     # 前端界面
├── uploads/           # 文件上传目录（自动创建）
└── package.json
```

## 🛡️ 安全说明

- 本应用设计用于受信任的本地网络
- 无需认证（同一网络内的任何人都可以加入）
- 上传的文件存储在服务器本地
